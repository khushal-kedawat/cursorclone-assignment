# Cursor Homepage Clone

A static clone of the Cursor.com homepage built with HTML and CSS.

## 📋 Sections Recreated

- **Navigation Bar** - Fixed navbar with logo, links, and buttons
- **Hero Section** - Main heading with CTA and hero image
- **Features** - Three feature cards with alternating layouts
- **Testimonials** - 6-column grid with quotes and profiles
- **Use Cases** - Three cards with images and descriptions
- **Changelog** - Four-column grid with recent updates
- **Highlights** - Two-column layout with article cards
- **CTA Section** - Large centered heading with download button
- **Footer** - Five-column navigation with copyright

## 🎨 Design System

### Typography
- **Font**: CursorGothic (custom)
- **Fallback**: system-ui, Helvetica Neue, Arial, sans-serif

### Colors
- **Backgrounds**: `#14120b` (main), `#1b1913` (cards), `#201e18` (elevated)
- **Text**: `#edecec` (primary), `rgba(237, 236, 236, 0.6)` (secondary)
- **Accent**: `#f54e00` (orange)

### Spacing
- Container max-width: `1300px`
- Section padding: `67px` vertical
- Border radius: `4px`

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Custom Web Fonts
